# SeeStaffKey - Плагин для управления режимом работы персонала на Minecraft сервере

## Описание

SeeStaffKey - это плагин для Minecraft серверов версии 1.16.5 и выше, предназначенный для удобного управления режимом работы администрации и модерации. Плагин позволяет персоналу активировать специальные режимы работы с помощью ключей, которые могут изменять группы игроков (через LuckPerms) или выполнять команды.

## Основные возможности

- Управление режимами работы персонала через команды
- Поддержка двух режимов работы: через LuckPerms или прямые команды
- Настройка индивидуальных ключей для разных членов команды
- Кастомизируемые сообщения и тайтлы
- Автоматическое снятие режима при выходе с сервера
- Защита от несанкционированного использования

## Установка

1. Скачайте последнюю версию плагина (SeeStaffKey-1.4.5-stable_latest.jar)
2. Поместите файл в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Настройте конфигурационный файл (`config.yml`) под ваши нужды

## Команды

### Основные команды (`/ssw`)
- `/ssw reload` - Перезагружает конфигурацию плагина
- `/ssw help` - Показывает справку по плагину
- `/ssw info` - Показывает информацию о плагине

### Команды управления ключами (`/key`)
- `/key [id ключа]` - Активирует режим работы с указанным ключом
- `/key stop [id ключа]` - Деактивирует режим работы
- `/key help` - Показывает справку по командам для персонала

## Настройка

Основные настройки находятся в файле `config.yml`:

```yaml
settings:
  mode: "lp" # Режим работы: "lp" (LuckPerms) или "command" (команды)
  not_found_keyid: "&cНеизвестный ключ !" # Сообщение о неизвестном ключе
  not_allowed_player: "&cВас нету в списке состава !" # Сообщение для неавторизованных игроков
  reload_message: "&aПлагин перезагружен !" # Сообщение о перезагрузке
  no_work_mode: "&cВы не находитесь в режиме работы !" # Сообщение при попытке выйти из неактивного режима

keys:
  key1: # ID ключа
    id: "example_id" # Описание ключа
    start_group: "modersw" # Группа при активации (для режима lp)
    stop_group: "moder" # Группа при деактивации (для режима lp)
    start_command: "/lp user {player} parent set modersw" # Команда при активации
    stop_command: "/lp user {player} parent set moder" # Команда при деактивации
    start_message: "&aВаш аккаунт успешно идентифицирован!" # Сообщение при активации
    stop_message: "&cВы вышли из режима работы!" # Сообщение при деактивации
    start_title: "&aВаш аккаунт подтверждён!" # Тайтл при активации
    stop_title: "&cВы вышли из режима работы!" # Тайтл при деактивации
    allowed_players: # Список игроков, имеющих доступ к этому ключу
      - "Player1"
      - "Player2"
```

## Требования

- Minecraft сервер версии 1.16.5 или выше
- Java 17 или выше
- Для режима LuckPerms требуется установленный плагин [LuckPerms](https://luckperms.net/)

## Разрешения

- `ssw.admin` - Доступ к командам `/ssw` (по умолчанию для операторов)
- `ssw.staff` - Доступ к командам `/key` (по умолчанию для операторов)
